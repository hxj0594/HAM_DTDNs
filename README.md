HAM_DTDNs

Code for the study:

The Global Economic Burden of Digestive Tract Diseases and Cancers from 2020 to 2050.

Overview

This repository contains the analytical code used to estimate the macroeconomic burden of 16 digestive tract diseases and neoplasms (DTDNs) across 190 countries and territories from 2020 to 2050 using a health-augmented macroeconomic modelling framework.

The study quantifies the long-term economic consequences of DTDNs by comparing a baseline scenario (observed disease burden) with a counterfactual scenario in which the target disease burden is eliminated. The modelling framework incorporates demographic structure, educational heterogeneity, labor force participation, disease-related mortality and morbidity, and the effects of treatment costs on physical capital accumulation.

Study aim

The main objective of this project is to estimate the future global and country-level economic burden attributable to DTDNs and to characterize how that burden varies:

* across diseases
* across countries and regions
* across income groups
* under different modelling assumptions and uncertainty scenarios

Main analytical components

This repository supports the following steps in the analysis pipeline:

1. Data preparation

   * harmonization of demographic, epidemiological, and macroeconomic inputs
   * preparation of disease-specific and country-specific model inputs

2. Main macroeconomic modelling

   * estimation of effective labor losses associated with mortality and morbidity
   * incorporation of treatment-cost-related effects on capital accumulation
   * comparison between baseline and counterfactual scenarios
   * calculation of GDP loss, GDP loss as a share of GDP, and per-capita economic loss

3. Imputation / estimation for missing countries

   * statistical imputation for countries with incomplete direct model inputs
   * generation of complete country coverage for aggregate outputs

4. Post-processing

   * aggregation of country-level and disease-specific results
   * generation of final tables and figures used in the manuscript and supplementary materials

Diseases included

The study covers 16 digestive tract diseases and neoplasms, including:

* Appendicitis
* Cirrhosis and other chronic liver diseases
* Colon and rectum cancer
* Esophageal cancer
* Gallbladder and biliary diseases
* Gallbladder and biliary tract cancer
* Gastritis and duodenitis
* Inflammatory bowel disease
* Inguinal, femoral, and abdominal hernia
* Liver cancer
* Pancreatic cancer
* Pancreatitis
* Paralytic ileus and intestinal obstruction
* Peptic ulcer disease
* Stomach cancer
* Vascular intestinal disorders

Repository contents

This repository may include:

* model scripts for the main macroeconomic analysis
* scripts for imputation and post-estimation processing
* helper scripts for result aggregation
* selected metadata or example inputs
* documentation for data preparation and reproducibility

To protect ongoing research development and because some inputs are derived from third-party data sources, this repository may not include all original raw datasets or all internal development scripts.

Reproducibility

The code in this repository is intended to reproduce the main analytical workflow and key aggregate outputs reported in the manuscript. Depending on data access and preprocessing requirements, users may need to:

* obtain raw data from the original public data providers
* place input files in the expected local folders
* run preprocessing scripts before running the main analysis scripts

Some results may depend on external source files that are not redistributed in full in this repository.

Software environment

The original analysis was implemented primarily in Python and/or R-based workflows (depending on the script), with standard data-analysis libraries for:

* data manipulation
* statistical modelling
* numerical computation
* figure/table generation

Users should install the packages required by each script before running the workflow.

Outputs

Typical outputs generated by the code include:

* annual country-level estimates
* aggregate country-level estimates
* disease-specific economic burden summaries
* imputed estimates for countries with missing direct model outputs
* tables and figures for the main text and supplementary materials

Intended use

This repository is provided for research transparency and reproducibility.
It is intended to help readers understand the analytical workflow used in the manuscript and to reproduce the main reported results where data access permits.
